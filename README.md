# Docker Rails Start App

Personal initial Rails configuration.

### Prerequisites

`ruby 2.5.0`
`nodejs 9.7.1`

Also checkout `.tool-versions` file.

### Installing

```
$ bundle install
```

## Built With

* [Rails](http://rubyonrails.org/)

* [Docker](https://www.docker.com/)
